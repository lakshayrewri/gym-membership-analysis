# Gym Membership Analysis

Excel and SQL project analyzing gym membership data with data cleaning, business queries, and interactive dashboard.

## Project Overview
This project analyzes gym membership data using SQL and Excel.  
The goal of this project was to clean raw data, solve business problems using SQL queries, and build an interactive Excel dashboard to present key insights.

## Tools Used
- SQL
- PostgreSQL
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

## Dataset Tables
The project includes the following tables:
- Members
- Trainers
- Classes
- Attendance
- Payments

## Project Workflow
### 1. Database Creation
Created relational tables for members, trainers, classes, attendance, and payments.

### 2. Data Cleaning
Performed data cleaning tasks such as:
- handling null values
- checking duplicate records
- fixing inconsistent membership type values
- identifying invalid ages and payment records
- validating foreign key relationships

### 3. Business Analysis
Solved business questions using SQL, including:
- monthly revenue analysis
- top revenue-generating members
- revenue by membership type
- active members by city
- average payment amount
- most popular classes
- trainer performance
- member attendance analysis
- top revenue-generating cities
- monthly new member growth

### 4. Dashboard Creation
Built an interactive Excel dashboard with:
- KPI cards
- slicers
- monthly revenue trend
- revenue by membership type
- top cities by revenue
- most popular classes
- business insights

## Key Insights
- Total Revenue: ₹8,54,000
- Total Members: 129
- Total Classes Offered: 20
- Monthly memberships generated higher revenue than yearly memberships
- December 2024 recorded the highest monthly revenue
- Functional Training was one of the most popular classes
- Noida was one of the top revenue-generating cities
